# Components

React component library providing reusable UI components and business components for common frontend development scenarios.

## Introduction

This section contains commonly used React components in the project, including basic UI components, business components, form components, etc. These components have been validated in production and can be used directly in projects.

## Component Categories

### Basic Components

Provides basic UI elements such as buttons, input fields, selectors, etc.

### Business Components

Components encapsulated for specific business scenarios, such as payment components, product display components, etc.

### Form Components

A collection of components for form scenarios, supporting data validation, linkage, and other features.
